# Banking System Database Design

This project focuses on designing a relational database system
for managing banking operations.

## Objectives
- Design ERD model for banking workflow
- Normalize database to Third Normal Form (3NF)
- Implement SQL queries for transactions and reporting
- Ensure ACID-compliant structure

## Key Skills
- Database Design
- SQL Development
- Data Modeling
- System Analysis

Full academic report included in this folder.
