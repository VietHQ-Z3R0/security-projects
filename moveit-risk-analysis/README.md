# MOVEit Data Breach 2023 – Risk Analysis

This project analyzes the MOVEit Transfer vulnerability (CVE-2023-34362)
using the NIST SP 800-30 Risk Assessment Framework.

## Objectives
- Identify vulnerability root cause
- Analyze threat actors and attack vectors
- Evaluate impact using CIA Triad
- Propose mitigation strategies

## Deliverable
Full academic report included in this folder.
